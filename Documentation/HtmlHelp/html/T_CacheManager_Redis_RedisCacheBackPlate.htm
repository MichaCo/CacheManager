RedisCacheBackPlate Class
Implementation of the cache back plate with Redis Pub/Sub feature.

Pub/Sub is used to send messages to the redis server on any Update, Cache Clear, Region Clear or Remove operation. Every cache manager with the same configuration subscribes to the same chanel and can react on those messages to keep other cache handles in sync with the 'master'.
