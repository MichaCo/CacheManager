CacheStatsT Class

Stores statistical information for a BaseCacheHandleTCacheValue.

Statistical counters are stored globally for the BaseCacheHandleTCacheValue and for each cache region!

To retrieve a counter for a region only, specify the optional region attribute of GetStatistics.
