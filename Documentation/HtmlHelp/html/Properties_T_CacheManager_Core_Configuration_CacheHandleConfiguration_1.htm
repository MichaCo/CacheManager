CacheName
Gets the name of the cache the handle got assigned to.
EnablePerformanceCounters
Gets a value indicating whether performance counters should be enabled or not.

If enabled, and the initialization of performance counters doesn't work, for example because of security reasons. The counters will get disabled silently.

EnableStatistics
Gets a value indicating whether statistics should be enabled.
ExpirationMode
Gets the expiration mode.
ExpirationTimeout
Gets the expiration timeout.
HandleName
Gets the name for the cache handle which is also the identifier of the configuration.
IsBackPlateSource
Gets a value indicating whether this instance is back plate source.

Only one cache handle inside one cache manager can be back plate source. Usually this is a distributed cache. It might not make any sense to define an in process cache as back plate source.

If no back plate is configured for the cache, this setting will have no effect.
